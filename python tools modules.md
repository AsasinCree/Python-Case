安装 pip
1. 打开 https://pip.pypa.io/en/latest/installing/ 把 "get-pip.py" 右键另存为下载下来
2. python get-pip.py (get-pip.py also installs setuptools and wheel if they are not already)
3. 添加 python 目录下的 Scripts 文件夹路径到环境变量
